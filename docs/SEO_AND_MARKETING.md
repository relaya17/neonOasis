# SEO ושיווק — NEON OASIS

## מה הוטמע בפרויקט (SEO טכני)

- **index.html:** כותרת בעברית, תיאור (meta description), מילות מפתח, Open Graph (פייסבוק/שיתופים), Twitter Card, canonical URL, robots.
- **robots.txt:** Allow /, Disallow /admin. עדכן את כתובת ה-Sitemap כשהאתר באוויר.

**חשוב:** אם הדומיין שלך שונה — החלף `https://neonoasis.com` בכל המקומות: **index.html** (canonical, og:url), **public/sitemap.xml** (כל ה־loc), **public/robots.txt** (Sitemap).

---

## המלצות SEO

### 1. דומיין ותוכן
- **דומיין:** שם קצר memorable (למשל neonoasis.io / neon-oasis.com).
- **SSL:** חובה (HTTPS) — משפיע על דירוג ואמון.
- **תוכן עמוד הבית:** כותרות H1/H2 ברורות, פסקאות עם המילים "משחקי מיומנות", "ששבש", "סנוקר", "פוקר", "לייב" — בלי לסתום, כדי שיהיה רלוונטי לחיפוש.

### 2. ביצועים (Core Web Vitals)
- דחיסת תמונות (WebP), lazy loading לווידאו/תמונות.
- Vite כבר עושה bundle ו־code splitting — שמור על גודל bundle סביר.
- אם יש הרבה תמונות: CDN או Cloudinary (כבר בשימוש לווידאו).

### 3. Sitemap ו־SSR (אם רוצים דירוג גבוה)
- **Sitemap.xml:** רשימת כל העמודים הציבוריים (/, /terms, /privacy, /leaderboard וכו') — ניתן ליצור סקריפט ב-build או ידנית.
- **SSR/Pre-render:** אפליקציית React היא SPA — למנועי חיפוש כדאי Pre-render (למשל prerender.io) או SSR (Next.js / Vite SSR) לעמוד הבית ולעמודי לנדינג, כדי שהתוכן יופיע ב־Google.

### 4. מילות מפתח (לא לחרוג)
- **להדגיש:** "משחקי מיומנות", "ששבש אונליין", "סנוקר", "פוקר", "לייב", "Skill-Based", "פדיון זכיות".
- **להימנע:** ביטויים שמזהים את האתר כהימורים/קזינו לא חוקי — התמקד ב־Skill-Based ומשחק אחראי.

### 5. רישום וניטור
- **Google Search Console:** רישום הדומיין, שליחת Sitemap, מעקב אחר שגיאות וקליקים.
- **Google Analytics 4 / Meta Pixel:** למדידת תנועה ומבצעים (בכפוף להסכמה במדיניות פרטיות).

## המלצות שיווק

### 1. קהל יעד
- חובבי ששבש, סנוקר, פוקר בישראל (ובשפה העברית).
- גיל 18+, מעוניינים במשחק מיומנות + אפשרות פדיון (לא רק "סושיאל קזינו").

### 2. ערוצים
- **אורגני:** SEO (כנ"ל), תוכן ברשתות (טיפים למשחק, הופעות לייב), פורומים/קבוצות ששבש ופוקר.
- **ממומן:** Google / Meta — רק בהתאם למדיניות הפרסום (משחקי מיומנות / Skill-Based; לא הימורים אסורים).
- **הפניה:** תוכנית Referral (כבר קיימת באפליקציה) — "הזמן חבר וקבל מטבעות".

### 3. מסרים
- "במקום שבו המזל נגמר, המוח מתחיל להרוויח" (כבר בסלוגן).
- הדגשת **מיומנות**, **לייב**, **מתנות מהקהל**, **פדיון חוקי** (Skill-Based).

### 4. אמון
- דף תנאי שימוש ומדיניות פרטיות (קיימים).
- דף משחק אחראי (קיים).
- הסבר מטבעות וארנק (מסמך בהסבר_מטבעות_וארנק.md).
- אפשר להוסיף באתר: "Certified Skill-Based Platform" / "גיל 18+".

## צעדים מעשיים מיידיים

1. **להחליף** ב־index.html את `https://neonoasis.com/` בכתובת האתר האמיתית.
2. **ליצור** sitemap.xml עם כל העמודים הציבוריים ולהוסיף את הכתובת ב־robots.txt.
3. **לרשום** את האתר ב־Google Search Console ולוודא שאין חסימות ב־robots.
4. **למדוד** ב־GA4 או כלי אחר: כניסות, מקורות, המרות (הרשמה, פדיון ראשון).
