# Cyber-Vegas 2.0 — מפרט עיצובי וטכני (UI/UX)

**מטרה:** להביא את הממשק לרמת אפליקציית על (High-End) — עתידני ויוקרתי, לא סתם רטרו.

---

## 1. השפה הוויזואלית: Cyber-Vegas 2.0

### אפקט ה-Glow (זוהר)
- **כל אלמנט אינטראקטיבי** (קובייה, קלף) **מטיל אור על הסביבה** ב-3D.
- דוגמה: קובייה ורודה מאירה את לוח השש-בש בורוד **בזמן שהיא מתגלגלת**.
- **דרישה טכנית:** שימוש ב-`PointLight` / `SpotLight` צמודים לאובייקט, או `emissive` + `emissiveIntensity` על ה-Material; עדכון מיקום האור לפי מיקום הקובייה/קלף בכל פריים.

### חומרים (Materials)
- **לוחות ושולחנות:** נראים כמו **זכוכית שחורה, שיש כהה או מתכת מוברשת** — קזינו יוקרתי בקומה ה-100.
- **דרישה טכנית:** `meshPhysicalMaterial` עם `roughness` נמוך, `metalness` גבוה, `envMapIntensity` (אם יש סביבה); צבעים כהים (`#0a0a0b`, `#1a1a1a`).

---

## 2. הפיד החברתי (The Neon Stream)

- **גלילה חלקה:** כמו בטיקטוק — מעבר בין "חדר" ל"חדר" ב-**Swipe למעלה** (scroll snap).
- **Live Preview:** בתוך הפיד רואים **משחקים אמיתיים** שקורים עכשיו ב-3D; **מצלמה קולנועית** (Cinematic Camera) נעה בזוויות שיוצרות אקשן.
- **אינטראקציה מהירה:** כפתור **"Join"** בולט בצבע ניאון — להצטרף למשחק או להמר על המנצח (טורניר פתוח).
- **בקוד היום:** `VegasFeed` — virtualizer, scroll snap, `GameScene` בתוך כל slide; כפתור Join/קישור ל-room — לחזק ויזואלית (ניאון, גודל).

---

## 3. ממשק הארנק והמטבע (Wallet UI)

- **עיצוב המטבע:** המטבע/צ'יפ הדיגיטלי מעוצב כ**צ'יפ הולוגרפי** שזז עם **חיישן התנועה** (Parallax Effect) — `deviceorientation` / `DeviceMotionEvent` או CSS `transform` לפי `tilt`.
- **גרפים של "הצלחה":** במקום רק רשימת עסקאות — **גרף ניאון** של עליית ערך הארנק לפי ניצחונות (עקומת צמיחה).
- **בקוד היום:** `Layout` מציג balance; `ProfileView` / `StoreView` — להוסיף אזור גרף (recharts עם ערכת צבעים ניאון) ופרזנטציה של Oasis/balance כ"נכס".

## 4. דרישות UX מהמתכנת

| דרישה | תיאור | יישום טכני |
|--------|--------|------------|
| **Haptic Feedback** | כשקוביות פוגעות בלוח או קלף מחולק — **רטט קטן ומדויק** (משחק פיזי). | `navigator.vibrate([10, 50, 10])` או Capacitor Haptics; לקרוא מתוך לוגיקת "dice landed" / "card dealt". |
| **Zero Loading** | **Skeleton Screens** — הממשק מופיע מיד; תוכן 3D נטען מאחורי **ערפל/טשטוש ניאון**. | מסך Splash / שלב ביניים עם שלד (מלבנים אפורים) במקום spinner; Canvas עם `Suspense` + fallback ערפל או blur. |
| **RTL/LTR Adaptive** | הממשק **מתהפך מושלם** בין עברית לאנגלית בלי לשבור עיצוב (חשוב לישראל). | `dir="rtl"` / `dir="ltr"` על root; `CacheProvider` עם RTL cache; `getTheme(direction)`; בדיקות ויזואליות ב-RTL. |

## 5. הסאונדטראק (Soundtrack)

| אלמנט | דרישה |
|--------|--------|
| **Ambient Synthwave** | מוזיקת רקע רגועה וקצבית — "טראנס" של המשחק. |
| **צלילי יוקרה** | צליל "קליק" מטבעות כמו **מטבעות זהב אמיתיים**, לא משחק זול. |
| **יישום** | אודיו ברקע (אופציונלי, כבוי כברירת מחדל); SFX על פעולות ארנק/ניצחון — קבצי אודיו קצרים; כיבוי/הפעלה בהגדרות. |

## 6. צ'ק-ליסט למתכנת/מעצב

- [ ] 3D: Glow דינמי על קוביות/קלפים (אור צמוד לאובייקט / emissive).
- [ ] 3D: חומרים — זכוכית שחורה, שיש כהה, מתכת מוברשת ללוחות.
- [ ] פיד: גלילה חלקה (scroll snap), כפתור Join ניאון בולט, Live Preview 3D עם מצלמה קולנועית.
- [ ] ארנק: צ'יפ הולוגרפי + Parallax; גרף ניאון של צמיחת ערך.
- [ ] UX: Haptic על "dice landed" / "card dealt".
- [ ] UX: Skeleton screens + טעינת 3D מאחורי ערפל/טשטוש.
- [ ] RTL/LTR: התהפכות מושלמת (כבר נתמך — לוודא בכל קומפוננטה).
- [ ] סאונד: Ambient Synthwave (אופציונלי); SFX יוקרתי למטבעות/ניצחון.
